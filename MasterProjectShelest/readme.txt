Написать программу для просмотра изображений.
Размер формы должен меняться соответственно размеру отображаемого изображения.

Реализовать меню:
File:
 Open - открыть изображение и отобразить его в PictureBox
 Close - закрыть изображение, сбросить PictureBox
 Exit - выйти из программы
Edit:
 Zoom in (+25%) - увеличить изображение на 25%
 Zoom out (+25%) - уменьшить изображение на 25%
 Reset size - сбросить размер изображения
About: вывести информацию о приложении и авторе в MessageBox.
